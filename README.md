# Prism Code Sentinel

Security-first AI code review with severity tags, exploit sketches, concrete fixes, and suggested tests. Uses **your OpenAI API key** only.

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · OpenAI Chat Completions (JSON mode)

## Run

```bash
npm install
npm run dev
```
